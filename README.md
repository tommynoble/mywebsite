# Thomas Boye Asante - Portfolio

A modern, clean portfolio website for Thomas Boye Asante, a UI/UX Designer specializing in Prompt Engineering and AI-enhanced design solutions.

## Features

- **Modern Design**: Clean, minimalist design with subtle gradients and excellent typography
- **Responsive**: Mobile-first responsive design that works on all devices
- **AI-Focused**: Specialized sections highlighting AI and Prompt Engineering expertise
- **Interactive**: Smooth animations and transitions throughout
- **Accessible**: Built with accessibility in mind using proper ARIA labels and semantic HTML

## Tech Stack

- **Framework**: Next.js 15 with App Router
- **Frontend**: React 19 with TypeScript
- **Styling**: TailwindCSS 4 with custom design tokens
- **UI Components**: shadcn/ui with Radix UI primitives
- **Icons**: Lucide React
- **Animations**: Framer Motion & CSS animations
- **Notifications**: Sonner for toast notifications

## Sections

1. **Hero Section**: Eye-catching introduction with gradient text and status badges
2. **About Section**: Two-column layout with professional photo and detailed bio
3. **Journey Section**: Timeline of professional experience with achievements
4. **Services Section**: AI-enhanced design services with pricing
5. **Case Studies**: Portfolio of successful projects with results
6. **Contact Section**: Professional contact form with consultation booking
7. **Footer**: Clean footer with contact info and social links

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

## Customization

The portfolio is built with flexibility in mind. Key customization areas:

- **Colors**: Modify the gradient and accent colors in `tailwind.config.ts`
- **Content**: Update personal information, experience, and projects in the respective component files
- **Images**: Replace placeholder images with actual photos and project screenshots
- **Contact**: Update contact information and social media links

## Design System

The portfolio uses a comprehensive design system with:

- **Typography**: Inter font with excellent hierarchy
- **Colors**: Light blue/cyan to white gradients with teal and purple accents
- **Spacing**: Generous white space following 8px grid system
- **Components**: Reusable UI components with proper TypeScript typing
- **Animations**: Subtle fade-in animations with staggered delays

## Performance

- Optimized images with Next.js Image component
- Proper semantic HTML for SEO
- Accessibility compliance with WCAG 2.1 guidelines
- Fast loading with modern build optimizations

## Deployment

The site is ready for deployment on platforms like:
